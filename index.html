Myntra Website Page

I developed a front-end clone of the Myntra website, focusing on replicating its visual design and layout structure. This project emphasizes accurate UI implementation, including product listings, banners, navigation menus, and overall styling inspired by the original platform.

The primary goal of this project was to strengthen my understanding of layout structuring, CSS styling, and building real-world interfaces using clean and organized code. This version is a static (non-responsive) implementation designed specifically for desktop view.

Key features

  • Sticky header with logo, navigation menu, search bar, and profile actions
  • Promotional homepage banners and category-based shopping sections
  • Product listing page with filter sidebar for category, brand, price, and color
  • Product detail page with breadcrumb trail, ratings, pricing, color options, size selection, and add-to-bag area
  • Floating free-shipping banner and notification bell UI
  • Footer with shopping links, customer policies, app promotion, social links, and trust indicators
  • Large asset organization with separate folders for banners, brands, categories, colors, and product images

What I Learned

  • How to build a realistic e-commerce front-end layout using only HTML and CSS
  • How to structure multi-page navigation for shopping websites
  • How to design product listing and product detail pages with clear information hierarchy
  • How to create sidebar filters, promotional sections, and category grids
  • How to manage large sets of assets and organize them cleanly in a project
  • How to improve visual consistency and spacing across multiple pages in a commercial-style UI
